# License-Plate-Detection-and-Recognition-using-OpenCV-and-Tesseract-OCR
**License Plate Detection and Recognition using OpenCV and Tesseract OCR -- is a computer vision project that detects vehicle license plates from images or videos and extracts the plate number using OCR. Built with Python, OpenCV, and Tesseract, it supports automated number plate recognition for traffic monitoring, parking, and security applications.

### Detailed GitHub Repository Description
**License Plate Detection and Recognition Using OpenCV and Tesseract OCR** is a computer vision project that automatically detects vehicle license plates from images or video streams and recognizes the alphanumeric characters using Tesseract OCR.

### Features
* Automatic license plate detection
* Image preprocessing for improved OCR accuracy
* License plate text extraction using Tesseract OCR
* Supports images and video input
* Fast and lightweight implementation with OpenCV
* Easy to customize and extend

### Tech Stack
* Python
* OpenCV
* Tesseract OCR
* NumPy
